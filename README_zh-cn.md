## 它可以帮你做：

允许你修改局域网世界的端口号，并选择是否映射这个端口使用UPnP映射到公网（如果你的路由器支持UPnP）。

允需其他玩家从任何地方连接你的世界。

会自动保存配置文件，并且下次加载世界时会自动载入配置。

你可以选择是否复制IP到剪切板。

根据你的游戏情况，自动选择设置。
